# 工单管理系统 - 用户手册

---

## 1. 系统简介

欢迎使用工单管理系统！

本系统旨在为您和您的团队提供一个清晰、高效、自动化的工作流程平台。它将复杂的项目流程分解为一个个独立的**工单**，并在不同角色（采购、摄影、美工、剪辑、运营、销售）之间无缝流转，确保每个环节都能准确、高效地完成。

**核心优势：**
- **流程自动化**：系统自动处理文件分发、状态更新和日志记录，减少手动操作。
- **职责清晰**：每个角色都有专属的操作界面，只关注与自己相关的任务。
- **进度透明**：所有工单的当前状态、历史记录一目了然，方便追踪。
- **操作简单**：直观的图形界面，无需复杂培训即可上手。

## 2. 快速上手

### 2.1. 启动与登录

1.  双击打开 `工单管理系统` 应用程序。
2.  在启动窗口中，选择您的**角色**和所属**部门**。
3.  点击"确定"，即可进入系统主界面。

![登录窗口](https://i.imgur.com/your-login-image.png)  <!-- 占位图，可替换 -->

### 2.2. 主界面导览

主界面是您最常用的工作区域，主要分为四个部分：

![主界面导览](https://i.imgur.com/your-main-window-image.png) <!-- 占位图，可替换 -->

1.  **部门筛选区 (左侧)**：列出了所有部门。点击某个部门，右侧的工单列表将只显示该部门的工单。默认显示"所有部门"。
2.  **工单列表区 (中间)**：以表格形式展示所有工单的核心信息，如工单ID、型号、名称、状态等。
    - **双击任意工单**，可以弹窗查看该工单的**完整详情和历史操作日志**。
    - **状态进度条**：直观显示工单当前在整个流程中的位置。
3.  **操作按钮区 (右下角)**：根据您登录时选择的角色，这里会显示您专属的操作按钮，如"采购办理"、"摄影办理"等。
4.  **通用功能区 (顶部/底部)**：
    - **创建工单**：发起一个新的工单流程（通常由"采购"角色使用）。
    - **刷新**：手动刷新工单列表，获取最新数据。
    - **任务管理器**：查看文件复制/移动的实时进度。

## 3. 核心功能详解

### 3.1. 查看工单详情与历史

在主界面的工单列表中，**双击任意一行**，系统会弹出一个窗口，详细展示该工单的所有信息和历史记录。

- **基础信息**：包含工单的ID、型号、名称、创建时间等。
- **产品信息**：由"运营"角色添加的产品标题、链接等（如果有）。
- **文件路径**：所有在流程中生成的文件夹路径，**双击路径即可直接在系统中打开对应的文件夹**。
- **操作日志**：记录了此工单从创建到现在的每一步操作，包括**操作人、时间、内容和IP地址**，方便追溯。

### 3.2. 任务管理器

当您进行"上传素材"、"分发文件"等涉及大量文件传输的操作时，系统会自动在后台处理，不会卡住您的界面。

您可以随时点击主界面右下角的 **`任务管理器`** 按钮，查看所有文件操作的实时进度。

- **进度条**：显示当前任务的完成百分比。
- **操作**：可以暂停、继续或取消正在进行的任务。

## 4. 各角色详细操作指南

> **通用操作**：在主界面选中一个或多个工单，然后点击右下角您角色对应的"办理"按钮，即可打开操作窗口。

### 4.1. 采购角色

**核心职责**：创建新工单，启动整个工作流程。

1.  在主界面点击 **`创建工单`** 按钮。
2.  在弹出的窗口中，填写以下信息：
    - **工单ID**：系统已自动生成，通常无需修改。
    - **产线/部门**：选择该产品属于哪个部门。
    - **型号** & **名称**：准确填写产品的型号和名称。
    - **发起人**：填写您的名字。
3.  点击"确定"，工单创建成功，状态自动变为 `拍摄中`。

### 4.2. 摄影角色

**核心职责**：上传原始素材，并将图片和视频分发给后期部门。

1.  在操作窗口中，首先**选择摄影师**。
2.  **上传素材**：点击 `上传素材` 按钮，选择您拍摄的所有原始图片和视频文件。系统会自动在服务器上创建规范的目录并上传。
3.  **分发图片/视频**：
    - 点击 `分发图片`，系统会自动将上传的素材中的**图片文件**复制到"美工待领取"目录。
    - 点击 `分发视频`，系统会自动将**视频文件**复制到"剪辑待领取"目录。
    - 工单状态自动更新为 `后期待领取`。

### 4.3. 美工 / 剪辑 角色

**核心职责**：从待领取目录获取素材，处理完成后，将成品分发给运营和销售。

1.  **领取素材**：点击 `领取素材` 按钮。系统会自动将"待领取"文件夹中的素材**移动**到您本地的"待处理"文件夹中，工单状态变为 `后期处理中`。
2.  **选择成品路径**：当您完成修图或剪辑后，将所有成品（图片、视频、源文件等）放在一个文件夹内。然后点击 `成品路径` 按钮，选择这个文件夹。
3.  **分发运营/销售**：
    - 点击 `分发运营`：系统会将您选择的成品路径下的**所有内容**复制到"运营部"的对应目录，但会**自动排除**任何名为 `源文件` 的文件夹。
    - 点击 `分发销售`：系统会将成品路径下的**所有内容**复制到"销售部"的对应目录，但会**自动排除**任何名为 `源文件`、`精修`、`详情页` 的文件夹。
    - 操作完成后，工单状态会自动更新为 `后期已完成`。

### 4.4. 运营角色

**核心职责**：领取后期完成的素材，并添加产品的上架信息。

1.  **选择存放路径**：首先点击 `选择存放路径` 按钮，选择一个您本地的文件夹，用于存放即将领取的素材。
2.  **领取素材**：点击 `领取素材` 按钮。系统会将美工/剪辑分发的成品**移动**到您指定的存放路径中。工单状态变为 `待上架`。
3.  **添加产品信息**：
    - 在下方的"产品上架信息"区域，填写产品的**标题、关键词和上架链接 (URL)**。
    - 您可以添加多条产品信息。
    - **双击已添加的URL**，可以直接在浏览器中打开链接。
4.  **完成上架**：当您完成所有上架工作后，请手动在主界面**双击工单**，在详情弹窗中将工单状态修改为 `已上架`。

### 4.5. 销售角色

**核心职责**：从素材中心获取最终的成品素材。

1.  **选择存放路径**：首先点击 `选择存放路径` 按钮，选择一个您本地的文件夹。
2.  **领取素材**：点击 `领取素材` 按钮。系统会将该工单所有可供销售的成品素材，从服务器的"销售部"目录**移动**到您指定的存放路径中。

## 5. 常见问题解答 (FAQ)

**Q: 点击"领取素材"或"分发"时，提示"文件/路径不存在"怎么办？**
**A:** 这个错误通常是因为您无法访问服务器上的共享文件夹。请按以下步骤排查：
    1.  **检查网络连接**：确保您的电脑网络正常。
    2.  **检查服务器/NAS连接**：在"访达"(macOS) 或"文件资源管理器"(Windows)中，确认您是否能正常打开 `Volumes` 或 `dabadoc` 等网络共享盘。如果无法打开，请联系IT管理员重新连接。
    3.  **路径变更**：确认对应的源文件夹是否被移动或重命名。

**Q: 进行文件传输时，程序界面没有反应（卡住了）？**
**A:** 请放心，程序并未卡住。所有文件操作都在后台进行。您可以随时点击主界面右下角的 **`任务管理器`** 按钮，查看详细的传输进度。这是一种防止界面假死的设计。

**Q: 如何理解不同的工单状态？**
**A:** 工单状态代表了它在整个流程中的生命周期：
    - **拍摄中**：工单已创建，等待摄影师上传素材。
    - **拍摄完成**：摄影师已上传素材，但还未分发。
    - **后期待领取**：摄影师已分发，等待美工/剪辑领取。
    - **后期处理中**：美工/剪辑已领取素材，正在处理。
    - **后期已完成**：美工/剪辑已处理完并分发给运营/销售。
    - **待上架**：运营已领取素材，等待上架。
    - **已上架**：运营已完成上架，流程结束。

**Q: 如果我错误地操作了一个工单怎么办？**
**A:** 系统会自动记录您的每一步操作。请及时联系管理员，在**工单详情**的历史日志中可以查看到所有操作记录，方便追溯和纠正。
